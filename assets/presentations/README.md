# PPT演示文件

本目录用于存放PPT演示文件，系统会自动识别并加载。

## 支持格式

- PDF（推荐）
- PPTX
- PPT

## 使用方法

1. 将PPT文件放入此目录，或通过前端界面上传
2. 在PPT控制页面选择要演示的文件
3. 启动摄像头，使用手势控制演示

## 手势控制映射

以下是默认的手势控制映射，可在系统设置中自定义：

| 手势 | 操作 | 说明 |
|------|------|------|
| 👎 Thumbs Down | 下一页 | 切换到下一张幻灯片 |
| 👍 Thumbs Up | 上一页 | 切换到上一张幻灯片 |
| ✋ Open（张开手） | 第一页 | 跳转到第一张幻灯片 |
| ✊ Close（握拳） | 最后一页 | 跳转到最后一张幻灯片 |
| ⬆️ Move Up | 向上滚动 | 在当前页面向上滚动 |
| ⬇️ Move Down | 向下滚动 | 在当前页面向下滚动 |

## 键盘快捷键

系统同时支持键盘快捷键操作：

- `→` 下一页
- `←` 上一页
- `Home` 第一页
- `End` 最后一页
- `Page Up` 向上滚动
- `Page Down` 向下滚动

## 注意事项

1. 建议使用PDF格式，浏览器兼容性更好
2. 文件大小建议不超过100MB
3. 确保文件名不包含特殊字符
