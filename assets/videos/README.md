# 视频文件

本目录用于存放视频文件，系统会自动识别并加载。

## 支持格式

- MP4（推荐）
- WebM
- AVI
- MKV
- MOV

## 使用方法

1. 将视频文件放入此目录，或通过前端界面上传
2. 在视频控制页面选择要播放的视频
3. 启动摄像头，使用手势控制播放

## 手势控制映射

以下是默认的手势控制映射，可在系统设置中自定义：

| 手势 | 操作 | 说明 |
|------|------|------|
| ✋ Open（张开手） | 播放 | 开始或继续播放视频 |
| ✊ Close（握拳） | 暂停 | 暂停视频播放 |
| 👌 OK | 重新开始 | 从头开始播放 |
| ✌️ Peace | 全屏 | 切换全屏模式 |
| 👍 Thumbs Up | 音量+10% | 增加音量 |
| 👎 Thumbs Down | 音量-10% | 降低音量 |
| ⬅️ Move Left | 后退10秒 | 快退10秒 |
| ➡️ Move Right | 前进10秒 | 快进10秒 |
| 🔄 Clockwise | 加速播放 | 提高播放速度（最高2x） |
| 🔃 Counter Clockwise | 减速播放 | 降低播放速度（最低0.25x） |

## 键盘快捷键

系统同时支持键盘快捷键操作：

- `Space` 播放/暂停
- `r` 重新开始
- `f` 全屏
- `↑` 音量+10%
- `↓` 音量-10%
- `←` 后退10秒
- `→` 前进10秒
- `>` 加速播放
- `<` 减速播放

## 注意事项

1. 建议使用MP4或WebM格式，浏览器兼容性最好
2. 文件大小建议不超过500MB
3. 确保文件编码格式为H.264（视频）+ AAC（音频）
4. 文件名不要包含特殊字符
